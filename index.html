Playwright — δοκιμή στο laptop

Άνοιξε PowerShell στο laptop και τρέξε. Χωρίς admin. Όλα κατεβαίνουν μέσω HTTPS.

Γρήγορος τρόπος one-liner

Κάνε τα πάντα με μία εντολή

irm https://raw.githubusercontent.com/gruncode/pw-lab/main/setup.ps1 | iex

Κατεβάζει portable Node + Playwright + Chromium, ανιχνεύει proxy & αρχιτεκτονική, και ανοίγει έναν ορατό browser για επιβεβαίωση. Τρέχει στη μνήμη → παρακάμπτει το ExecutionPolicy.

Εναλλακτικά — χειροκίνητα

1. Portable Node

cd $env:TEMP
$arch = if ($env:PROCESSOR_ARCHITECTURE -eq 'ARM64') {'arm64'} else {'x64'}
$ver='v20.17.0'; $dir="node-$ver-win-$arch"
Invoke-WebRequest -UseBasicParsing "https://nodejs.org/dist/$ver/$dir.zip" -OutFile node.zip
Expand-Archive -Force node.zip .
& ".\$dir\node.exe" -v

2. Playwright + δοκιμή

$node=".\$dir\node.exe"; $npm=".\$dir\npm.cmd"
'{ "name":"pwlab" }' | Out-File -Encoding ascii package.json
& $npm i playwright --no-fund --no-audit
& $node node_modules\playwright\cli.js install chromium
'const{chromium}=require("playwright");(async()=>{const b=await chromium.launch({headless:false});const p=await b.newPage();await p.goto("https://www.bing.com");console.log("TITLE:",await p.title());await p.waitForTimeout(4000);await b.close();console.log("OK works");})().catch(e=>{console.error("FAIL:",e.message)})' | Out-File -Encoding ascii test.js
& $node test.js
Τι ψάχνουμε
  • OK works + ανοίγει παράθυρο browser → ✅ τρέχει.
  • Download fail → proxy/CDN μπλοκάρει.
  • Node δεν τρέχει / "blocked by policy" → AppLocker/WDAC — πες μου.
